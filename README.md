# react-native-parity-blur

Cross-platform native backdrop blur for React Native (Fabric)

## Installation


```sh
npm install react-native-parity-blur
```


## Usage


```js
import { ParityBlurView } from "react-native-parity-blur";

// ...

<ParityBlurView color="tomato" />
```
